한글 문서 생태계의 패러다임 전환

기술 생태계 심층 분석

Py

Python: 데이터 자동화의 중추

python-hwpx & hwpxskill

한컴오피스 의존성 없이 순수 파이썬으로 OWPML을 제어합니다. 특히 fill_by_path 기능을 통해 AI가 템플릿에 데이터를 주입하는 코드를 단 몇 줄로 생성할 수 있게 돕습니다.

RAG 파이프라인 최적화

hwp-parser는 LangChain과 연동되어 수천 개의 문서를 Vector DB로 삽입하는 기업용 지식 베이스 구축의 핵심 도구로 기능합니다.

Rs

Rust & WASM: 고성능 시각화

rhwp (알, 모두의 한글)

조판 엔진을 Rust로 재구현하여 브라우저에서 HWPX를 SVG로 렌더링합니다. WASM 덕분에 클라이언트 사이드에서 보안 위협 없이 빠른 미리보기가 가능해졌습니다.

HwpForge: 무손실 JSON 라운드트립

HWPX를 JSON으로 변환해 AI가 편집하게 한 뒤, 다시 완벽한 HWPX로 재조립하는 계층형 아키텍처를 제공하여 데이터 무결성을 보장합니다.

바이브 코딩(Vibe Coding) 실전 가이드

AI 에이전트(Claude Code, Cursor)에게 권한을 부여하고, 자연어로 문서를 조작하는 구체적인 방법론입니다.

PROMPT

시나리오: 대량 문서 양식 채우기

"폴더의 '신청서_양식.hwpx'를 읽고, 엑셀의 신청인 명단을 기반으로 각각의 문서를 생성해 줘. treesoop/hwp-mcp 도구를 사용해서 작업 결과물을 SVG로 미리보기 보여줘."
✔ 사용 도구: treesoop/hwp-mcp ✔ 핵심: 시각적 피드백 루프
SETUP

MCP 서버 즉시 연동 (Claude Code)

claude mcp add hwp-mcp -- npx -y hwp-mcp@latest

이 명령어 한 줄로 AI 에이전트에게 77개 이상의 HWPX 제어 도구를 장착할 수 있습니다.

⚠

보안 경보: CVE-2026-7599

위협 내용: MCP 서버 내 경로 탐색(Path Traversal) 취약점 발생.

원인: output_path 인자에 대한 화이트리스트 검증 부재.

위험성: 환각 상태의 AI가 시스템 파일을 덮어쓰거나 파괴할 수 있음.

대응 전략 (MAC):

  • 에이전트 권한 최소화 (Root 금지)
  • 샌드박스(AppArmor/SELinux) 적용
  • 인간 승인(Human-in-the-loop) 정책
