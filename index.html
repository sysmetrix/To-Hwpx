왜 To HWPX인가

AI 시대의 작업물을 한글 행정 문서로 연결합니다

AI와 함께 일하면 초안은 MD·DOCX·HTML로, 데이터는 CSV·JSON으로, 분석은 노트북(.ipynb)으로 더 빠르게 만들어집니다만 공공 행정의 제출 환경에서는 여전히 한글 문서(hwpx)가 필요할 때가 많습니다.
한글로, To HWPX는 새롭게 만들어지는 작업 형식을 한글 행정 문서로 연결해 반복 복사·붙여넣기와 오래 걸리는 서식 재작업을 줄입니다.

PC와 모바일 브라우저 안에서 7단계로 완전히 처리됩니다

1. 파일 선택

드래그앤드롭 또는 클릭으로 변환할 파일을 선택합니다.

2. 포맷 파싱

확장자를 자동으로 감지하고 포맷별 파서로 IR(중간 표현)으로 변환합니다.

3. HWPX 생성

IR → section0.xml → JSZip 패키징으로 HWPX 파일을 생성합니다.

4. 구조 검증

ZIP_STORED 규칙, 네임스페이스, 참조 무결성 등 4개 영역을 자동 검증합니다.

5. 다운로드

검증을 통과한 .hwpx 파일을 즉시 다운로드합니다.

입력 포맷별 현재 변환 상태입니다. ★★★: 구조 보존 양호 / ★★☆: 기본 구조 변환 / ★☆☆: 내용 추출 중심

포맷 품질 상태 파싱 방식 주요 제한사항
MD (Markdown) ★★★ 지원됨 marked.js 토큰 분석 + GFM 각주·frontmatter · WebP 변환 → IR 상대경로·접근 차단 이미지 제외 가능
DOCX ★★★ 베타 JSZip → document.xml + numbering.xml + rels → IR 페이지 배치·스타일 테마·주석·변경 추적 손실 가능

CSV ★★☆ 베타 RFC 4180 파서 → 표 IR 없음
XLSX ★★☆ 베타 SheetJS → 첫 시트 → IR 첫 번째 시트만 변환, 셀 서식 대부분 손실
JSON ★★☆ 지원됨 JSON.parse → 구조 분석 → IR 보고서형 레이아웃 자동 설계 없음
TXT ★★★ 지원됨 줄바꿈 분석 → IR 없음

폰트 설치 안내

웹 화면의 글꼴 미리보기와 한글(HWPX) 출력은 별개입니다. 선택한 폰트를 한컴오피스에서 그대로 보려면 PC에 TTF/OTF 폰트를 설치해야 합니다.

설치 후 브라우저 재시작 없이 바로 '다시 확인'을 누르세요.
폰트 설치 방법
  1. TTF 파일 내려받기 → Shift 키 누르고 파일 우클릭 → 모든 사용자용으로 설치
  2. 이 창에서 다시 확인 클릭 (권한 요청 팝업이 뜨면 허용)
  3. 한컴오피스에서 사용하려면 한컴오피스 재시작
도움말

대부분의 문서는 아래 순서만 기억하면 됩니다.

  1. 파일 선택 드롭존에 입력 파일을 올립니다.
  2. 기본 설정 확인 글꼴, 줄 간격, 용지만 가볍게 봅니다.
  3. 변환 후 다운로드 HWPX를 한컴에서 열어 최종 확인합니다.

본문 서식 처리
원본 우선은 원본(DOCX·HTML·XLSX)의 서식을 먼저 따르고, 설정 우선은 아래 옵션을 강하게 적용합니다. 혼합은 그 절충입니다. MD·TXT·CSV·JSON처럼 원본 서식이 없는 입력은 항상 아래 설정대로 만들어집니다.
문단 앞/뒤 간격
문단 사이 여백을 좁게/기본/넓게로 조정합니다. 줄글이 많은 문서는 기본 또는 크게가 읽기 편합니다.
제목 스타일
제목 크기·굵기를 조절합니다. 보고서는 큰·굵게, 메모는 작게가 어울립니다.
표 스타일
기본은 전체 테두리, 단순은 옅은 테두리, 머리행 음영은 첫 행에 회색 배경을 넣습니다.
링크 표시
파랑+밑줄은 일반 하이퍼링크, 검정 본문은 본문색 링크, 텍스트+주소는 표시 글자 뒤에 실제 주소를 함께 적습니다(인쇄용).
이미지 최대 폭 / 정렬
이미지를 본문의 50%·75%·100%까지로 제한하고 왼쪽·가운데·오른쪽으로 배치합니다. 원본에 정렬 정보가 있으면 그쪽이 우선합니다.
첫 제목 본문 처리
상단 제목 블록으로 올라간 첫 제목을 본문에서 제거할지 유지할지 정합니다.

폰트

HWPX에는 폰트 파일이 들어가지 않고 이름만 기록됩니다. 다른 PC에서 같은 모양을 원하면 같은 폰트를 설치해야 합니다.

보존 한계

DOCX나 HTML의 복잡한 원본 레이아웃을 그대로 복제하는 도구가 아니라, 내용을 한글 문서 구조로 정리하는 변환기입니다.

추천 순서

먼저 기본값으로 변환한 뒤, 필요할 때만 여백과 제목 블록을 조정하는 흐름이 가장 빠르고 안정적입니다.

앞으로 추가하기 좋은 세부 설정은 문단 앞/뒤 간격, 제목 스타일 프리셋, 표 스타일, 링크 표시 방식, 이미지 최대 폭/정렬입니다.
