PC와 모바일 브라우저 안에서 7단계로 완전히 처리됩니다

📥

1. 파일 선택

드래그앤드롭 또는 클릭으로 변환할 파일을 선택합니다.

🔄

2. 포맷 파싱

확장자를 자동으로 감지하고 포맷별 파서로 IR(중간 표현)으로 변환합니다.

⚙️

3. HWPX 생성

IR → section0.xml → JSZip 패키징으로 HWPX 파일을 생성합니다.

✅

4. 구조 검증

ZIP_STORED 규칙, 네임스페이스, 참조 무결성 등 4개 영역을 자동 검증합니다.

📦

5. 다운로드

검증을 통과한 .hwpx 파일을 즉시 다운로드합니다.

★★★: 구조 보존 양호 / ★★☆: 기본 구조 변환 / ★☆☆: 내용 추출 중심

포맷 품질 상태 파싱 방식 주요 제한사항
MD (Markdown) ★★★ 지원됨 marked.js → HTML → IR 이미지 미지원
HTML ★★☆ 지원됨 DOMParser → IR CSS 스타일 손실, 이미지 미지원
DOCX ★☆☆ 지원됨 JSZip → word/document.xml → IR 이미지·머리글·바닥글·복잡한 서식 미지원
HWP ★☆☆ 베타 ZIP 시도 → 텍스트 추출 HWP5(바이너리) 파싱 제한적
TXT ★★★ 지원됨 줄바꿈 분석 → IR 없음
CSV ★★☆ 지원됨 RFC 4180 파서 → 표 IR 없음
XLSX ★★☆ 지원됨 SheetJS → 첫 시트 → IR 첫 번째 시트만 변환, 셀 서식 대부분 손실
JSON ★★★ 지원됨 JSON.parse → 구조 분석 → IR IR 형식이면 직접 사용
IPYNB ★★☆ 지원됨 JSON 파싱 → cell_type별 처리 이미지 출력 셀 미지원
PC / 모바일 브라우저 ★★★ 지원됨 브라우저 로컬 처리 iOS/Android 저장 UI는 브라우저 정책을 따름
PDF ★★☆ 예정 — 레이아웃 추출 복잡도로 백엔드 필요
PPTX ★☆☆ 예정 — 슬라이드→문서 변환 시 정보 손실 큼
ODT / RTF ★★☆ 예정 — 복잡한 바이너리/XML 파싱
EPUB ★★☆ 예정 — EPUB3 구조 파싱 복잡도

📄 HWPX 미리보기

rhwp 뷰어를 불러오는 중...

최초 실행 시 WebAssembly 로드로 10~20초 소요될 수 있습니다

기본 미리보기는 문서 구조를 브라우저 안에서 바로 렌더한 것으로, 한컴오피스의 글꼴·여백·표 너비와 다를 수 있습니다. 정확한 결과는 다운로드 후 한컴오피스에서 확인하세요. '정밀 미리보기'는 외부 rhwp 뷰어(WebAssembly)를 사용합니다.
PC 브라우저 지원
파일 선택 → HWPX 변환 → 다운로드

Chrome, Edge, Safari, Firefox 등 최신 PC 브라우저에서 파일 선택, 변환, HWPX 다운로드 흐름을 지원합니다.

  • 변환은 서버 업로드 없이 브라우저 안에서 처리됩니다.
  • 드래그앤드롭과 클릭 파일 선택을 모두 사용할 수 있습니다.
  • 자동 다운로드가 차단되면 완료 카드의 HWPX 다운로드 버튼으로 다시 받을 수 있습니다.

Android Chrome과 iPhone Safari에서 파일 선택, 변환, HWPX 다운로드 흐름을 지원하도록 구성되어 있습니다.

  • Android는 Chrome 다운로드 항목에 .hwpx 파일로 저장되는 흐름을 지원합니다.
  • iPhone은 Safari 다운로드 또는 파일 앱 저장 흐름을 사용합니다.
  • 모바일 브라우저 정책에 따라 자동 다운로드가 막히면 완료 카드의 다운로드 버튼을 다시 누르면 됩니다.
앱처럼 설치하기
to-hwpx 설치
HWPX → 홈 화면

Chrome

  1. 주소창 오른쪽의 설치 아이콘을 누릅니다.
  2. 아이콘이 없으면 메뉴 ⋮ 를 누릅니다.
  3. 앱 설치 또는 홈 화면에 추가를 선택합니다.

Edge

  1. 주소창 오른쪽의 앱 설치 아이콘을 누릅니다.
  2. 아이콘이 없으면 메뉴 ⋯ 를 누릅니다.
  3. 앱 > 이 사이트를 앱으로 설치를 선택합니다.
안전한 로컬 처리

파일은 PC와 모바일 브라우저에서만 처리됩니다. 서버에 전송되지 않습니다.

  • 업로드한 파일은 변환 과정에서 사용자 기기(브라우저) 안에서만 읽고 처리됩니다.
  • 원본 파일도, 변환된 HWPX도 외부 서버로 전송·저장되지 않습니다.
  • 변환 결과는 사용자가 직접 내려받으며, 페이지를 닫으면 메모리에서 사라집니다.
  • 단, '정밀 미리보기'를 사용할 때만 외부 뷰어(rhwp)를 불러오며, 이때도 파일은 브라우저 안에 머뭅니다.
폰트 설치 안내

웹 화면의 글꼴 미리보기와 한글(HWPX) 출력은 별개입니다. 선택한 폰트를 한컴오피스에서 그대로 보려면 PC에 TTF/OTF 폰트를 설치해야 합니다.

설치 후 브라우저 재시작 없이 바로 '다시 확인'을 누르세요.
폰트 설치 방법
  1. TTF 파일 내려받기 → Shift 키 누르고 파일 우클릭 → 모든 사용자용으로 설치
  2. 이 창에서 다시 확인 클릭 (권한 요청 팝업이 뜨면 허용)
  3. 한컴오피스에서 사용하려면 한컴오피스 재시작
